addDays

Returns a new Date instance with a date the specified number of days from the provided date.

addDays(date: (Date | number), days: number): Date
Parameters
date ((Date | number)) The date to modify.
days (number) The number of days to move from the provided date.
Returns
Date: The result is a new Date instance the specified number of days from the provided date.
Throws
  • TypeError: A TypeError is thrown when the arguments are not of the expected type.

addMonths

Returns a new Date instance with a date the specified number of months from the provided date. The day of the month (DotM) will be the last day of the month in case the original DotM doesn't exist in the result month.

addMonths(date: (Date | number), months: number): Date
Parameters
date ((Date | number)) The date to add months to.
months (number) The number of months to move from the provided date.
Returns
Date: Returns a new Date instance the specified number of months from the provided date.
Throws
  • TypeError: A TypeError is thrown when the arguments are not of the expected type.
Example
// Add 1 month to January 31, 2019
addMonths(new Date(2019, 0, 31), 1);
// => February 28, 2019

createRange

Creates an array of sequential Date instances.

createRange(startDate: (Date | number), length: number): Array<Date>
Parameters
startDate ((Date | number)) The start date of the range.
length (number) The number of days to return. When it is a negative number the range will start in the past and the last day in the range will be the start date.
Returns
Array<Date>: Returns an array with the specified number of Date instances.
Throws
  • TypeError: A TypeError is thrown when the provided values are not of type Date or Number.

createRangeForDays

Creates an array of sequential Date instances.

createRangeForDays(startDate: (Date | number), endDate: (Date | number)): Array<Date>
Parameters
startDate ((Date | number)) The start date of the range (inclusive).
endDate ((Date | number)) The date when the range ends (inclusive).
Returns
Array<Date>: Returns an array with Date instances, starting at the start date all the way to the end date.
Throws
  • TypeError: A TypeError is thrown when the provided values are not of type Date or Number.
  • RangeError: A RangeError is thrown when the provided end date is earlier than the start date.

getFirstDayOfWeek

getFirstDayOfWeek(date: (Date | number), firstWeekDay: number): Date
Parameters
date ((Date | number)) The date for which to return the first day of the week.
firstWeekDay (number = 1) Optional parameter to specify on which day a week starts. By default this will be Monday.

isEarlier

Test if the provided date falls before a maximum date. The comparison is made including the time component of the date.

isEarlier(date: (Date | number), maxDate: (Date | number), isInclusive: boolean): boolean
Parameters
date ((Date | number)) The date to check against a maximum date.
maxDate ((Date | number)) The maximum date to compare against.
isInclusive (boolean = true) Optional parameter to specify whether the max date is considered a valid date. By default this is true, the max date itself is considered to be valid.
Returns
boolean: Returns true when the provided date is earlier than the provided max date.
Throws
  • TypeError: A TypeError is thrown when the arguments are not of the expected type.
Example
// This wil return true
  const date = new Date(2019, 0, 1);
  isEarlier(date, date);
  // This will return false
  isEarlier(date, date, false);

Returns
